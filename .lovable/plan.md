

## Brand Identity: **MedScreen**

A premium SaaS platform for medical digital signage — powering screens in hospitals, clinics, waiting rooms, and dental offices.

---

### Brand Name & Tagline
- **Name:** MedScreen
- **Tagline:** "Smarter screens for better care."

### Logo Concept
- A minimal, geometric mark combining a screen/monitor silhouette with a subtle medical cross or pulse line
- Clean wordmark in a modern sans-serif, with the icon usable standalone as a favicon

### Color Palette
- **Primary:** Deep teal (#0D9488 range) — medical trust, modern tech feel
- **Secondary:** Navy/slate (#1E293B) — authority, professionalism
- **Accent:** Soft sky blue (#38BDF8) — approachability, clarity
- **Success/CTA:** Vibrant teal-green for action buttons
- **Neutrals:** Cool grays for text hierarchy and backgrounds
- **Background:** Clean white with subtle cool-gray sections

### Typography
- **Headings:** Inter (700/600) — clean, geometric, premium
- **Body:** Inter (400) — excellent readability
- **Mono/accents:** JetBrains Mono for any code/tech details

### Visual Identity Principles
- Generous whitespace, card-based layouts with subtle shadows
- Rounded corners (8-12px) for a friendly-professional feel
- Subtle gradient accents (teal → sky blue) for premium depth
- Medical imagery style: abstract, geometric — no stock photos of doctors
- Icon style: Lucide icons, consistent stroke weight

### Implementation
1. Set up CSS custom properties with the full color palette in HSL
2. Configure Tailwind with brand colors, typography, and spacing
3. Create a branded layout shell (header with logo + nav, footer with brand)
4. Build a simple brand showcase page displaying the palette, typography, and logo
5. Ensure dark mode variables are also defined for consistency
